Olá, eu sou o José Baptista, professor, Psicólogo e desenvolvidor web.

Desde aí comecei uma aventura nunca imaginada por por mim e descobri que para tal tinha que aprender linguages de programação, como html, css e javascript, python etc. Foi aí que comecei uma jornada imparável até se tornar um desenvolvidor web.
